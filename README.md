# AI Local System

Docker compose for deploying an AI application set for a local system. The following container deployment will deploy from compose.

- HAproxy (localhost port 80 / 443)
- n8n Workflow
- Postgres SQL
- REDIS memory DB
- Milvus vector store

<u>Required</u>
Requires BIOS level virtualization enabled and install windows linux subsystem (WSL).
